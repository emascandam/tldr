# eopkg

> Package manager used in Solus Linux distribution.

- Search the repo for packages:

`eopkg search "{{search_pattern}}"`

- Install a package with its required dependencies:

`eopkg install {{package_name}}`

- Remove a package:

`eopkg remove {{package_name}}`

- Perform a full system & packages upgrade:

` eopkg upgrade`

- Upgrade a specific package and its dependencies:

` eopkg upgrade {{package_name}}

- Show package and maintainer details:

`eopkg blame {{package_name}}`
